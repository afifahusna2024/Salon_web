### Deploy to Netlify

1. Push your code to GitHub
2. Connect your repository to Netlify
3. Netlify will automatically detect Next.js and use the `netlify.toml` configuration
4. Your site will be deployed automatically!

### Deploy to Vercel (Recommended for Next.js)

1. Push your code to GitHub
2. Import your repository in Vercel
3. Vercel will automatically detect Next.js settings
4. Deploy!

### Deploy to GitHub Pages (Static Export)

If you need to deploy as static files:

1. Update `next.config.mjs` to use `output: 'export'`
2. Run `npm run build`
3. Deploy the `out` folder to GitHub Pages
